doxygen
=======
Please see templates folder for sample resources

`doxygen::generate`
-------------------
- generates documentation with `doxygen` for all files in `DOXYGEN_DOXY_FILES`


Dependencies
============
- `utils`
- `version`

3rd party:
- `doxygen` - http://www.doxygen.nl


Examples
========
```
$ bee doxygen::generate
```
